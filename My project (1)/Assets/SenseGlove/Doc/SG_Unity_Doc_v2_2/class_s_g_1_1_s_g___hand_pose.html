Member Data Documentation

◆ jointAngles

Vector3 [][] SG.SG_HandPose.jointAngles

The angles of each joint, in degrees, where flexing the finger creates a negative z-rotation. The first index [0..4] determines the finger (thumb..pinky), while the second [0..2] determines joint (CMC, MCP, IP for thumb. MCP, PIP, DIP for fingers.)

◆ jointPositions

Vector3 [][] SG.SG_HandPose.jointPositions

The position of each joint, in meters, relative to a Wrist Transform: (JointPosition * WristRotation) + WristPosition = 3D Position. The first index [0..4] determines the finger (thumb..pinky), while the second [0..2] determines joint (CMC, MCP, IP for thumb. MCP, PIP, DIP for fingers.)

◆ jointRotations

Quaternion [][] SG.SG_HandPose.jointRotations

The quaternion rotation of each joint, relative to a Wrist Transform: JointRotation * WristRotation = 3D Rotation. The first index [0..4] determines the finger (thumb..pinky), while the second [0..2] determines joint (CMC, MCP, IP for thumb. MCP, PIP, DIP for fingers.)

◆ normalizedFlexion

float [] SG.SG_HandPose.normalizedFlexion

The total flexion of each finger, normalized to values between 0 (fingers fully extended) and 1 (fingers fully flexed). The index [0..4] determines the finger (thumb..pinky).
