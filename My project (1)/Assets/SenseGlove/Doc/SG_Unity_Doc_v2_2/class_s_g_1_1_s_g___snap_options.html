Detailed Description

Add this to a SG_Grabable to snap the hand at a fixed location (and pose?), either on Hover or OnGrab.

It is placed in a separate behaviour so I can extend off it's GenerateGrabArgs / GetSnapPoint functionality. (For example, allowing you to assign a whole list of SnapPoints and choosing the closest.
