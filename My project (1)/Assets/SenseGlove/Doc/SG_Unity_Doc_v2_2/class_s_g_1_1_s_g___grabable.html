Detailed Description

An object that can be moved around using one or more hands. When held by a GrabScript, it calculates a target position and rotation, then moves towards it. Grabables override the wrist position- and rotation, drawing the hand on the spot you grabbed it.

Member Enumeration Documentation

◆ DualHandMode

enum SG.SG_Grabable.DualHandMode
strong

How the rotation of a Grabable is determined when held with two hands.

Enumerator
FreezeRotation 

The Grabable stays in the same rotation it was when the second hand joined in. Not realistic, but allows for quick transfers between hands.

AverageRotation 

Calculates the target rotation for the two hands, then halves it. Works well enough, but has a tendency to flip if when rotating both hands in opposite directions.

LeftHandRotation 

Use the target rotation of the left hand.

RightHandRotation 

Use the target rotation of the right hand.

FirstHandsRotation 

Use the target rotation of whichever hand held on first (note; this changes as you swap hands).

PivotPoints 

Treat the two refrences as pivots.
