Detailed Description

A Gameobject that despawns an objects once its material breaks, and optionally replaces it with a 'broken' version.

enum SG.SG_Breakable.UnbreakType
strong

How the object will respond after it breaks.

Enumerator
None 

The object stays broken, and does nothing. Default value.

Unbreak 

The object unbreaks after the timer elapses.

Reset 

The object fully resets after the timer elapsed.
