enum SG.SG_CalibrationSequence.StartCondition

When to start a calibration sequence.

Enumerator
Manual 

Calibration sequence will ony start when we call A StartCalibration or NextCalibrationStep function.

WhenNeeded 

Only start calibration when the HapticGlove determines it must be done (you're not in the same range as last time).

OnConnected 

This calibration will always run when the glove (re)Connects
