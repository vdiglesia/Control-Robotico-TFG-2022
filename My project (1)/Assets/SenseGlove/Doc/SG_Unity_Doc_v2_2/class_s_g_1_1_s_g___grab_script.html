◆ heldObjectLimit

int SG.SG_GrabScript.heldObjectLimit = 1

How many objects a GrabScript can hold onto at the same time. Ideally, it should stay at 1, because holding two objects that determine position could result in undefined behaviour.

If set to less than 0, it's ignored. If set to 0, you won't be able to grab something.
