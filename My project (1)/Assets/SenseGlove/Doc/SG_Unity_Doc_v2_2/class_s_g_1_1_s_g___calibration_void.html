enum SG.SG_CalibrationVoid.VoidStage

Enumerator
WaitForStart 

Waits for "Begin()" to be called before launching into calibration

WaitingForFirst 

Wait until you've got the first glove.

WaitingForSecond 

Then, we wait X seconds for another Glove to connect.

Introduction 

Then, we start calibration with either 1 or 2 gloves. We start with some instructions

MoveFingers 

Collect data while the user moved their finger. Give them some extra encouragement

ConfirmCalibration 

We have enough data to compile profiles, but are waiting for the user to confirm with a thumbs up

Done 

We're finished. Compile profiles, congratulate the user, and move on.

NoCalibrationError 

When something goes wrong
