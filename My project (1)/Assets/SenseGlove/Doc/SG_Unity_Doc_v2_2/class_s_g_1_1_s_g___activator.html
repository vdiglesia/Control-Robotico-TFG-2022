Detailed Description

An all-purpose script to toggle multiple GameObjects, Renderers and/or Monobehaviour Scripts at will.

Split into three different components because they do not all share the 'enabled' logic. You can extend from this script to add an auto-assign on Awake() or Start() and save yourself some manual assigning.
