Detailed Description

A base class that can be overriden. It fires events and keeps track of hands, but does not manipulate the object it's attached to.

Use this script to use make an object recognizeable by an SG_GrabScript.

Member Enumeration Documentation

◆ GrabMethod

enum SG.SG_Interactable.GrabMethod
strong

Grab conditions for an Interactable, restricting grabbing based on certain contitions

Enumerator
Any 

This object can be grabbed by one or two hands

OneHandOnly 

This object can only be grabbed by one hand.

RightHandOnly 

This object can only be grabbed by the right hand

LeftHandOnly 

This object can only be grabbed by the left hand

None 

This object cannot be grabbed by itself. However, its GrabPoints can still be grabbed.
