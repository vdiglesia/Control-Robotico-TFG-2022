Detailed Description

A class implementing this Interface can provide one with a SG_HandPose. We don't care how or where it comes from.

float SG.IHandPoseProvider.OverrideGrab ( )

Returns true if this script wants to grab objects regardless of HandPose. Useful when you have it mapped to a button.

Is a float so we can determine between a light and hard grasp. If it's a simple button, just use 0 and 1.

Returns
